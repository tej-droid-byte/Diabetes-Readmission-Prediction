# 🩺 Diabetes Readmission Prediction & Healthcare Analytics Dashboard

Machine Learning and Healthcare Analytics project focused on predicting diabetes patient readmission risk using hospital encounter records from 130 US hospitals.

This project combines **Healthcare AI**, **Machine Learning**, and **Business Intelligence** techniques to analyze readmission patterns, patient demographics, diagnosis distributions, medication behavior, and hospitalization trends.

The project includes predictive analytics and interactive Power BI dashboards for healthcare decision support.

# 📌 Project Description

Hospital readmissions significantly impact healthcare cost, patient outcomes, and hospital resource allocation.

This project investigates diabetes patient encounters to understand:

✅ Readmission behavior

✅ Patient demographics

✅ Hospital stay trends

✅ Diagnosis distributions

✅ Medication influence

✅ Risk factors associated with readmission

---

# 🗂 Dataset Information

Dataset Source:

Diabetes Hospital Readmission Dataset

Dataset Characteristics:

- 101K+ patient encounters
- 130 US hospitals
- Time Period: 1999–2008

Key Variables:

- Patient demographics
- Age
- Gender
- Race
- Diagnoses
- Medication history
- Hospital stay duration
- Readmission status
