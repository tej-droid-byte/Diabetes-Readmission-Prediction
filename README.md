# Diabetes Readmission Prediction and Analytics Dashboard

## Overview
Machine learning and Power BI analytics project for predicting diabetes patient readmission using hospital encounter records from 130 US hospitals.

## Dataset
- 101K+ patient encounters
- 130 US hospitals
- Period: 1999–2008
- Features: demographics, diagnosis, medication history, hospital stay

## Dashboard Pages

### Executive Dashboard
- Patient KPIs
- Readmission distribution
- Age group analysis
- Hospital stay analytics

### Risk Factor Dashboard
- Diagnosis distribution
- Medication load vs hospital stay
- Race analysis
- Gender readmission analysis

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Power BI
- Jupyter Notebook
